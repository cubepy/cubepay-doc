## [1.3.0]

### Added
- عضویت اجباری در کانال تلگرام برای استفاده از ربات
- احراز هویت شماره تلفن با اشتراک‌گذاری کانتکت واقعی تلگرام (نه تایپ دستی)
- دکمه‌ی «منوی اصلی» در همه‌ی بخش‌ها
- راهنمای دانلود و پیکربندی اپ فورواردر پیامک (با آدرس اختصاصی هر فروشنده)

### Fixed
- اسکیپ نشدن اسم کسب‌وکار/کارت در پیام‌های HTML تلگرام که می‌تونست پیام رو کلاً خراب کنه
- بخش «مدیریت فروشنده‌ها» یه ستون قدیمی و بلااستفاده رو برای کارت نشون/ویرایش می‌کرد

---

## [1.2.0]

### Added
- چند کارت برای هر فروشنده + حالت چرخش (دستی / رندوم / ساعتی)
- کارمزد اختصاصی قابل‌تنظیم برای هر فروشنده (جدا از کارمزد پیش‌فرض پلتفرم)
- شارژ خودکار کیف‌پول از داخل ربات (بدون نیاز به واریز دستی خارج از ربات)
- اعلان تلگرامی مستقیم به فروشنده بعد از هر تراکنش موفق (شامل مبلغ، کارمزد، موجودی جدید)
- هشدار پیشگیرانه‌ی موجودی کم کیف‌پول
- آمار کلی تراکنش‌ها برای ادمین (امروز/هفته/ماه/کل)
- مدیریت کامل فروشنده‌ها از ربات (ویرایش، تغییر وضعیت، حذف)

### Fixed
- رزرو کارمزد به‌صورت اتمیک شد تا کیف‌پول در شرایط همزمانی بالا منفی نره
- جلوگیری از ثبت‌نام تکراری با قفل یکتا روی آیدی تلگرام

### Security
- جلوگیری از SSRF روی آدرس کال‌بک فروشنده‌ها
- محدودیت نرخ درخواست روی وب‌هوک پیامک
