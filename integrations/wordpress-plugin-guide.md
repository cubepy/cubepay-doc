# 🔌 افزونه‌ی ووکامرس کیوب‌پی (کارت‌به‌کارت + ارز دیجیتال)

اگه فروشگاه شما با **وردپرس + ووکامرس** ساخته شده، به‌جای اتصال دستی به API، می‌تونید از این افزونه‌ی آماده استفاده کنید — نصبش چند دقیقه‌ست و نیازی به کدنویسی ندارید.

---

## ✅ پیش‌نیاز

- وردپرس + **ووکامرس** فعال روی سایتتون
- PHP 8.1 به بالا (اکثر هاست‌های امروزی دارن)
- حساب فروشندگی‌تون تو ربات مدیریت فروشنده‌ها (`@cubepy_bot`) باید **تایید شده** باشه

---

آخرین نسخه‌ی افزونه رو از اینجا دانلود کنید (فایل zip):

**[⬇️ دانلود smspay-gateway.zip](https://github.com/cubepy/cubepay-doc/releases/latest/download/smspay-gateway.zip)**

---

## 🚀 نصب (۳ قدم)

### ۱) بارگذاری افزونه

تو پیشخوان وردپرس: **افزونه‌ها ← افزودن افزونه ← بارگذاری افزونه**، فایل zip دانلودشده رو انتخاب و **نصب** کنید، بعد **فعال‌سازی** بزنید.

### ۲) گرفتن اطلاعات از ربات تلگرام

تو ربات `@cubepy_bot`، از منوی «🔗 پنل من» این مورد رو کپی کنید:
- **توکن API**
- **آدرس سرور کارت‌به‌کارت** (شبیه `https://cubevps.ir/smspay`) — همیشه لازمه
- 🆕 اگه می‌خواید ارز دیجیتال هم قبول کنید: **آدرس سرور یکپارچه** (شبیه `https://cubevps.ir/pay`)

### ۳) تنظیم افزونه

تو وردپرس برید به: **ووکامرس ← تنظیمات ← پرداخت ← «CubePay» ← مدیریت**

- ✅ فعال‌سازی رو بزنید
- 📋 توکن API و آدرس سرور کارت‌به‌کارت رو پیست کنید
- 🆕 **آدرس سرور یکپارچه (اختیاری):** اگه پر کنید و رو حساب فروشندگی‌تون (تو ربات، «⚙️ روش‌های پرداخت») ارز دیجیتال هم فعال باشه، مشتری موقع پرداخت بین کارت و کریپتو انتخاب می‌کنه. اگه خالی بذارید، دقیقاً مثل قبل فقط کارت‌به‌کارت کار می‌کنه.
- 💰 **واحد پول فروشگاه** رو درست انتخاب کنید (تومان یا ریال) — این خیلی مهمه، چون درگاه فقط ریال/تومان مشخص قبول می‌کنه و اگه اشتباه انتخاب بشه مبلغ ۱۰ برابر یا یک‌دهم می‌شه!
- ذخیره کنید

تمام! حالا تو صفحه‌ی پرداخت فروشگاهتون، گزینه‌ی پرداخت برای مشتری‌ها نمایش داده می‌شه.

---

## 🧪 تست کنید

یک سفارش آزمایشی با مبلغ کم بسازید و مسیر پرداخت رو کامل تا آخر برید — باید بعد از واریز (چه کارتی چه کریپتویی)، خودکار به صفحه‌ی «سفارش شما تکمیل شد» ووکامرس برگردید.

---

## ❓ مشکل داشتید؟

- مطمئن بشید حساب فروشندگی‌تون تو ربات **تایید شده** و **کارت بانکی** ثبت کرده باشید (بدون کارت، درگاه کارتی فاکتور نمی‌سازه).
- برای ارز دیجیتال، مطمئن بشید هم تو ربات «کریپتو» رو فعال کردید، هم آدرس سرور یکپارچه رو تو تنظیمات افزونه پر کردید.
- اگه پرداخت انجام شد ولی سفارش تو ووکامرس «پرداخت‌شده» نشد، از پشتیبانی کیوب‌پی بخواید لاگ سفارش (Order notes) رو چک کنه.
- برای هر مشکل دیگه: 🎫 [پشتیبانی کیوب‌پی](https://t.me/cube_sup)
