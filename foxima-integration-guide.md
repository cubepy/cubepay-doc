# 🔌 راهنمای اتصال CubePay به ربات Foxima (یا فورک‌هاش)

این راهنما برای فروشنده‌هاییه که ربات فروش VPN شون رو با کد **Foxima** (یا هر فورک/نسخه‌ی مشابهی که همین ساختار رو داره) ساختن. اگه فروشنده‌ای بگه «ربات من مثل مال شماست»، این فایل رو مستقیم بهش بدید.

⏱ زمان تخمینی برای یه برنامه‌نویس PHP معمولی: **کمتر از ۱ ساعت**

---

## قبل از شروع، این ۳ تا رو از ربات CubePay (پنل من) داشته باشید:

1. 🔑 **API Token** (از دکمه‌ی «🔗 پنل من»)
2. 🌐 **آدرس create-payment** (چیزی شبیه: `https://YOUR-CUBEPAY-DOMAIN/smspay/api/create-payment.php`)
3. 🌐 **آدرس verify-payment** (چیزی شبیه: `https://YOUR-CUBEPAY-DOMAIN/smspay/api/verify-payment.php`)

---

## فایل ۱ — `re/rx/function/business_logic_1.php`

دنبال تابع `createPayZarinpey` بگردید. **۲ خط** رو پیدا و عوض کنید:

**پیدا کنید:**
```php
$ch = curl_init('https://zarinpay.me/api/create-payment');
```
**تغییر بدید به:**
```php
$ch = curl_init('https://YOUR-CUBEPAY-DOMAIN/smspay/api/create-payment.php');
```

توکن رو نیازی به تغییر تو کد نیست — از تنظیمات خود ربات (پنل مدیریت → درگاه‌های ریالی → توکن) وارد می‌کنید.

⚠️ **حتماً پسوند `.php` رو فراموش نکنید** — برخلاف zarinpay.me که آدرس تمیز داره، فایل واقعی رو سرور CubePay این پسوند رو داره.

---

## فایل ۲ — `payment/ZarinPay/successful.php`

دنبال این خط بگردید:

**پیدا کنید:**
```php
$ch = curl_init('https://zarinpay.me/api/verify-payment');
```
**تغییر بدید به:**
```php
$ch = curl_init('https://YOUR-CUBEPAY-DOMAIN/smspay/api/verify-payment.php');
```

بعدش، چند خط پایین‌تر، دنبال این تیکه بگردید و **کامل حذفش کنید** (یا با `//` کامنتش کنید):
```php
$verifyCode = $result['data']['code'] ?? null;
if (!in_array($verifyCode, [100, 101], true)) {
    throw new Exception('پرداخت تایید نشد.');
}
```
📌 این چک مخصوص فرمت پاسخ zarinpay.me بود. CubePay این فیلد رو نداره و اصلاً لازمش نیست — چون چند خط بالاتر پاسخ `success` رو از قبل چک کرده.

---

## فایل ۳ — تنظیمات ربات (بدون نیاز به کد)

تو پنل مدیریت ربات (یا از طریق چت ادمین)، بخش «درگاه‌های ریالی» → «زرین‌پی»:
- ✅ فعال‌سازی رو روشن کنید
- 🔑 توکنی که از CubePay گرفتید رو بذارید
- (اختیاری) اسم دکمه رو از «زرین پی» به هر چیزی که دوست دارید تغییر بدید

---

## ✅ تست کنید

1. تو ربات خودتون، یه شارژ کیف‌پول کوچیک (مثلاً ۱۰,۰۰۰ تومان) امتحان کنید
2. لینک پرداخت رو باز کنید، مبلغ دقیق (با آفست چند تومانی) رو ببینید
3. کارت‌به‌کارت کنید
4. چند ثانیه صبر کنید — باید پیام «پرداخت موفق» بگیرید و کیف‌پول ربات شما شارژ بشه

اگه به مشکلی خوردید که تو این راهنما نبود، لاگ‌های زیر معمولاً کافیه که مشکل رو پیدا کنیم:
- `error_log` خود ربات (کنار فایل‌های PHP)
- گزارش خطا (اگه ربات کانال گزارش خطا داره، دنبال «خطا در ساخت لینک» بگردید)
