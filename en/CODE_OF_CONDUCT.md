# Contributor Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of age, experience level, nationality, or any other
characteristic.

## Expected Behavior

- Using respectful and inclusive language
- Respecting differing viewpoints and experiences
- Accepting constructive criticism gracefully

## Unacceptable Behavior

- Offensive language or imagery
- Personal or political attacks
- Public or private harassment

## Enforcement

Violations can be reported via [@cubepy_bot](https://t.me/cubepy_bot).
