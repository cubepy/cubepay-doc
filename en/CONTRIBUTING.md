# Contributing to CubePay Docs

Thanks for wanting to help improve this documentation 🙏

## How do I contribute?

1. Fork the repo.
2. Create a new branch: `git checkout -b fix/typo-in-readme`
3. Make your changes.
4. Open a Pull Request describing what you fixed/added.

## What can you improve?

- Fixing typos or grammar
- Adding code samples for other languages
- Clarifying confusing explanations
- Reporting errors in the documentation (not in the service itself)

## What doesn't belong here

- Reporting bugs in the CubePay service (the API itself) → via [@cubepy_bot](https://t.me/cubepy_bot)
- Reporting security vulnerabilities → follow [SECURITY.md](./SECURITY.md)

## Simple rules

- Be respectful and constructive (see [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md)).
- For large changes, open an issue first so we can align.
