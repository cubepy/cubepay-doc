# مشارکت در CubePay Docs

از اینکه می‌خواید در بهبود این مستندات کمک کنید ممنونیم 🙏

## چطور مشارکت کنم؟

1. ریپو رو Fork کنید.
2. یک برنچ جدید بسازید: `git checkout -b fix/typo-in-readme`
3. تغییرات رو اعمال کنید.
4. Pull Request باز کنید و توضیح بدید چه چیزی رو اصلاح/اضافه کردید.

## چه چیزهایی رو می‌تونید بهبود بدید؟

- رفع اشتباهات تایپی یا گرامری
- اضافه کردن نمونه کد برای زبان‌های دیگه
- بهبود توضیحات مبهم
- گزارش خطا در مستندات API (نه در خود سرویس)

## چیزهایی که اینجا جاشون نیست

- گزارش باگ سرویس CubePay (خود API) → از طریق [@cubepy_bot](https://t.me/cubepy_bot)
- گزارش آسیب‌پذیری امنیتی → طبق [SECURITY.md](./SECURITY.md)

## قوانین ساده

- محترمانه و سازنده باشید (به [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) نگاه کنید).
- برای تغییرات بزرگ، قبلش یک Issue باز کنید تا هماهنگ بشیم.
