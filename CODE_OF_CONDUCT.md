# آیین رفتاری مشارکت‌کنندگان

## تعهد ما

ما متعهدیم که مشارکت در این پروژه برای همه، صرف‌نظر از سن، سطح تجربه،
ملیت یا هر ویژگی دیگری، تجربه‌ای بدون آزار باشد.

## رفتار مورد انتظار

- استفاده از زبان محترمانه و فراگیر
- احترام به دیدگاه‌ها و تجربیات متفاوت
- پذیرش انتقاد سازنده

## رفتار غیرقابل قبول

- استفاده از زبان یا تصاویر توهین‌آمیز
- حمله‌ی شخصی یا سیاسی
- آزار عمومی یا خصوصی
